# Multi-Touch Marketing Attribution & ROI Dashboard

## Project Overview
This project analyzes customer journeys across multiple marketing channels and builds attribution models to measure campaign effectiveness and marketing ROI.

## Objectives
- Analyze customer touchpoints across marketing channels
- Implement First-Touch Attribution
- Implement Last-Touch Attribution
- Implement Linear Attribution
- Calculate KPIs (ROAS, CAC, CPC)
- Build an interactive Power BI Dashboard

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Power BI
- Jupyter Notebook
- Git & GitHub
